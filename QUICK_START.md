# 快速开始指南

## 🚀 5 分钟上手

### 步骤 1：安装依赖并启动

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问：http://localhost:3000

---

### 步骤 2：配置 Gemini API（必须）

首次打开会看到 "🔴 API 未配置"

1. 点击 **"立即配置"** 按钮
2. 在弹出的配置面板填写：

```
API 地址：https://your-proxy.com/v1
API 密钥：AIza... (您的 Gemini API Key)
文本模型：gemini-2.5-pro (默认)
图像模型：gemini-2.5-flash-image (默认)
```

3. 点击 **"保存配置"**
4. 看到 "🟢 API 已配置" 即成功

---

### 步骤 3：生成第一个 PPT

#### 输入脚本

在左侧输入框填写：

```
本节课介绍人工智能的三大分支：
1. 机器学习：让计算机从数据中学习规律
2. 深度学习：模仿人脑神经网络的学习方式
3. 强化学习：通过奖励机制优化决策
```

#### 上传模板（可选）

**支持 3 种方式**：

1. **粘贴截图**（推荐）
   - 在 PowerPoint 中截图 (截屏或 Win+Shift+S)
   - 在模板上传区域按 `Ctrl+V` 粘贴

2. **拖拽图片**
   - 将图片拖拽到上传区域

3. **点击上传**
   - 点击上传区域选择图片文件

#### 点击生成

点击 **"开始生成"** 按钮，等待处理完成（约 30-60 秒）

---

### 步骤 4：下载结果

生成完成后，右侧会显示：

- **参考预览图** - Gemini 生成的精美效果图
- **可编辑 PPT** - 真实的 .pptx 文件，文字不乱码
- **配图需求** - 提取的配图描述和关键词

点击 **"下载全部"** 或单独下载

## 📝 使用建议

### ✅ 推荐做法

1. **脚本简洁明了**
   - 每页 PPT 控制在 100-300 字
   - 使用要点式，不要长段落

2. **提供模板截图**
   - 截图完整的 PPT 页面
   - 包含配色和样式参考

3. **验证生成结果**
   - 参考图用于视觉参考
   - 可编辑 PPT 用于实际编辑
   - 对照修改，减少返工

### ❌ 避免问题

1. **脚本过长** - 单页超过 500 字会影响效果
2. **无标题主题** - 至少要有明确的主题
3. **忘记配置 API** - 必须先配置才能使用

## 🔧 常见问题

### Q: 为什么生成失败？

**A:** 检查以下几点：
1. API 配置是否正确（测试连接）
2. 网络是否畅通（代理是否可用）
3. 脚本是否太长或为空
4. 检查浏览器控制台错误信息

### Q: 生成的 PPT 可以编辑吗？

**A:** 可以！生成的是真实的 .pptx 文件：
- ✅ 文字可编辑、复制
- ✅ 样式可修改
- ✅ 图片可替换
- ✅ 支持 PowerPoint/WPS/Keynote

### Q: 参考图和 PPT 有什么区别？

**A:** 两者互补使用：
- **参考图** - Gemini 生成的视觉效果图，可能有文字乱码，仅供参考
- **可编辑 PPT** - 真实 PPT 文件，文字准确，但设计受限于代码生成

外包人员应该：**参照参考图的视觉效果，在可编辑 PPT 基础上调整**

### Q: 模板截图必须是 .pptx 文件吗？

**A:** 不需要！支持任何图片格式：
- ✅ 截图（PNG, JPG）
- ✅ 照片
- ✅ 扫描件
- ❌ 不支持直接上传 .pptx 文件

### Q: 如何批量处理多个页面？

**A:** 批量功能开发中，暂时请：
1. 单页面逐个生成
2. 或等待下次更新

## 💡 Pro Tips

1. **快捷粘贴模板**
   - 在 PPT 中按 `Alt+PrtSc` 截图当前页
   - 在上传区域按 `Ctrl+V` 粘贴
   - 无需保存图片文件

2. **重复使用配置**
   - API 配置自动保存
   - 同一台电脑只需配置一次

3. **保存历史记录**
   - 生成记录自动保存在浏览器
   - 刷新页面不会丢失

4. **提高生成质量**
   - 脚本要点化、数字化
   - 提供清晰的模板参考
   - 明确页面主题
